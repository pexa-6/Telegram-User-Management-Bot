## ✨ Features



- **👥 Automatic User Recording** - Save user data on first message

- **📊 Database Management** - CRUD operations for users

- **📨 Mass Messaging** - Send messages and files to all or selected users

- **🔍 Search and Filter** - Find users by username or name

- **🛡️ Admin Panel** - Full control for administrator

- **📝 Action Logging** - Detailed logging of all operations

- **🚫 Error Handling** - Proper exception handling

- **🔄 Auto-restart** - Autonomous operation with restart on errors



## 🏗️ Architecture



### Technology Stack

- **Python 3.13** - Main programming language

- **pyTelegramBotAPI** - Library for Telegram Bot API

- **SQLite3** - Lightweight database for user storage

- **python-dotenv** - Configuration management via environment variables

- **Logging** - Python's built-in logging system
